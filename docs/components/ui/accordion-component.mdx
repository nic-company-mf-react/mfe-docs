# Accordion 컴포넌트

`mfe-lib-shared`에서 제공하는 아코디언 컴포넌트입니다.  
각 항목을 클릭하면 내용이 펼쳐지거나 닫힙니다. 아래 플레이그라운드에서 **코드를 직접 수정**하면 미리보기가 실시간으로 반영됩니다.

---

## 임포트

```ts
import {
  Accordion,
  AccordionItem,
  AccordionTrigger,
  AccordionContent,
} from '@nic/mfe-lib-shared/components/ui';
```

---

## 다중 열기 (Multiple)

`multiple` prop을 추가하면 여러 항목을 동시에 열 수 있습니다.

```jsx live
function MultipleDemo() {
  return (
    <Accordion multiple defaultValue={['multi-1', 'multi-3']}>
      <AccordionItem value="multi-1">
        <AccordionTrigger>알림 설정</AccordionTrigger>
        <AccordionContent>
          이메일 알림, 푸시 알림, SMS 알림을 개별적으로 설정할 수 있습니다.
          알림 빈도와 수신 시간대도 지정 가능합니다.
        </AccordionContent>
      </AccordionItem>
      <AccordionItem value="multi-2">
        <AccordionTrigger>개인정보 및 보안</AccordionTrigger>
        <AccordionContent>
          2단계 인증을 활성화하고 로그인 기록을 확인하세요.
          연결된 앱과 기기를 관리할 수 있습니다.
        </AccordionContent>
      </AccordionItem>
      <AccordionItem value="multi-3">
        <AccordionTrigger>결제 및 구독</AccordionTrigger>
        <AccordionContent>
          청구서 조회, 결제 수단 변경, 구독 취소를 이 섹션에서 관리하세요.
        </AccordionContent>
      </AccordionItem>
    </Accordion>
  );
}
```

## Disabled (비활성화)

`AccordionItem`에 `disabled` prop을 추가하면 해당 항목을 비활성화할 수 있습니다.

```jsx live
function DisabledDemo() {
  return (
    <Accordion>
      <AccordionItem value="enabled-1">
        <AccordionTrigger>계정 이용 내역을 볼 수 있나요?</AccordionTrigger>
        <AccordionContent>
          마이페이지 &gt; 이용 내역에서 최근 30일간의 활동을 확인할 수 있습니다.
        </AccordionContent>
      </AccordionItem>
      <AccordionItem value="disabled-1" disabled>
        <AccordionTrigger>프리미엄 기능 정보 (비활성화)</AccordionTrigger>
        <AccordionContent>
          이 항목은 프리미엄 사용자에게만 제공됩니다.
        </AccordionContent>
      </AccordionItem>
      <AccordionItem value="enabled-2">
        <AccordionTrigger>이메일 주소는 어떻게 변경하나요?</AccordionTrigger>
        <AccordionContent>
          계정 설정 &gt; 프로필에서 이메일 주소를 변경할 수 있습니다.
          변경 후 인증 이메일이 발송됩니다.
        </AccordionContent>
      </AccordionItem>
    </Accordion>
  );
}
```

## 인터랙티브 플레이그라운드

열린 항목 상태를 직접 확인하면서 아코디언을 조작해보세요.

```jsx live
function Playground() {
  const [openItems, setOpenItems] = React.useState([]);
  const [isMultiple, setIsMultiple] = React.useState(false);

  const controlStyle = {
    display: 'flex',
    flexDirection: 'column',
    gap: '8px',
    marginBottom: '16px',
    fontSize: '13px',
  };
  const rowStyle = {
    display: 'flex',
    alignItems: 'center',
    gap: '8px',
  };
  const labelStyle = {
    width: '100px',
    fontWeight: 'bold',
    color: '#555',
  };

  return (
    <div style={{ fontFamily: 'sans-serif' }}>
      {/* 컨트롤 패널 */}
      <div style={{
        background: '#f8fafc',
        border: '1px solid #e2e8f0',
        borderRadius: '8px',
        padding: '16px',
        marginBottom: '20px',
      }}>
        <div style={controlStyle}>
          <div style={rowStyle}>
            <span style={labelStyle}>multiple</span>
            <input
              type="checkbox"
              checked={isMultiple}
              onChange={e => {
                setIsMultiple(e.target.checked);
                setOpenItems([]);
              }}
            />
            <span style={{ fontSize: '12px', color: '#94a3b8' }}>
              여러 항목을 동시에 열 수 있습니다
            </span>
          </div>
          <div style={rowStyle}>
            <span style={labelStyle}>열린 항목</span>
            <span style={{ fontSize: '12px', color: '#1e293b', fontWeight: 'bold' }}>
              {openItems.length === 0 ? '없음' : openItems.join(', ')}
            </span>
          </div>
        </div>
      </div>

      {/* 미리보기 */}
      <div style={{
        padding: '24px',
        background: '#fff',
        border: '1px dashed #cbd5e1',
        borderRadius: '8px',
      }}>
        <Accordion
          multiple={isMultiple}
          value={openItems}
          onValueChange={setOpenItems}
        >
          <AccordionItem value="item-1">
            <AccordionTrigger>배송 옵션은 무엇인가요?</AccordionTrigger>
            <AccordionContent>
              일반 배송(5~7일), 빠른 배송(2~3일), 당일 배송을 제공합니다.
              국제 주문은 무료 배송입니다.
            </AccordionContent>
          </AccordionItem>
          <AccordionItem value="item-2">
            <AccordionTrigger>환불 정책이 어떻게 되나요?</AccordionTrigger>
            <AccordionContent>
              구매 후 30일 이내에 미사용 상태의 제품을 반품하시면 전액 환불해 드립니다.
              반품 운송비는 고객 부담입니다.
            </AccordionContent>
          </AccordionItem>
          <AccordionItem value="item-3">
            <AccordionTrigger>고객 지원에 연락하려면 어떻게 하나요?</AccordionTrigger>
            <AccordionContent>
              이메일(support@example.com), 전화(평일 오전 9시 ~ 오후 6시),
              또는 채팅을 통해 문의하실 수 있습니다.
            </AccordionContent>
          </AccordionItem>
        </Accordion>
      </div>
    </div>
  );
}
```

## Props

### Accordion (Root)

| Prop | 타입 | 기본값 | 설명 |
|---|---|---|---|
| `multiple` | `boolean` | `false` | 여러 항목을 동시에 열 수 있는지 여부 |
| `defaultValue` | `string[]` | `[]` | 초기에 열려 있을 항목의 `value` 배열 (비제어) |
| `value` | `string[]` | - | 현재 열린 항목 (제어 컴포넌트) |
| `onValueChange` | `(value: string[]) => void` | - | 열린 항목 변경 시 호출되는 핸들러 |
| `className` | `string` | - | 루트 요소에 추가할 CSS 클래스 |

### AccordionItem

| Prop | 타입 | 기본값 | 설명 |
|---|---|---|---|
| `value` | `string` | - | 항목을 식별하는 고유 값 **(필수)** |
| `disabled` | `boolean` | `false` | 항목 비활성화 여부 |
| `className` | `string` | - | 추가할 CSS 클래스 |

### AccordionTrigger

| Prop | 타입 | 기본값 | 설명 |
|---|---|---|---|
| `children` | `ReactNode` | - | 트리거에 표시될 내용 **(필수)** |
| `className` | `string` | - | 추가할 CSS 클래스 |

### AccordionContent

| Prop | 타입 | 기본값 | 설명 |
|---|---|---|---|
| `children` | `ReactNode` | - | 패널에 표시될 내용 **(필수)** |
| `className` | `string` | - | 추가할 CSS 클래스 |
